# Code of Conduct

This project follows the [Go Community Code of Conduct](https://go.dev/conduct).

All participants in issues, pull requests, discussions, and any other project
spaces are expected to abide by it.

## Reporting

Report concerns privately to **john.anto.young@gmail.com**. Do not open public
issues for conduct reports. See [SECURITY.md](./SECURITY.md) for vulnerability
reporting.
