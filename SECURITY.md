# Security Policy

## Supported versions

While go-mcf is 0.x, only the latest minor release receives security fixes.

| Version | Supported          |
| ------- | ------------------ |
| 0.x     | Latest minor only  |
| < 0.x   | No                 |

## Reporting a vulnerability

Report vulnerabilities privately via [GitHub Security Advisories](https://github.com/branched-services/go-mcf/security/advisories/new).

Do **not** open public issues or pull requests for vulnerabilities. Public
conduct concerns belong in [CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md).

## Response SLA

- **Acknowledgement:** within 14 days of a valid report.
- **Disclosure target:** within 90 days of acknowledgement, coordinated with
  the reporter. Critical issues may be disclosed sooner once a fix is
  available.
